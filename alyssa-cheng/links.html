Design Games

hi, i've compiled a few links to some online design games / exercises! please enjoy (◕‿◕)

  • It's Centred That: train your eye to centre items

  • The Font Game: what fonts can you recognize?
  • Better Web Type: adjust type attributes according to best practices for web type
  • What The Hex?: choose the corresponding colour to the hex code provided
