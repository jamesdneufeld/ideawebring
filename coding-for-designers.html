Guiding and supporting the web ring community.

  • James Neufeld

Why the web ring format? This project integrates multiple course learning outcomes and my pedagogical approach:

  • Internet History - Experience the pre-social web through web rings
  • HTML/CSS Practice - Build real sites with professional workflows
  • Front-End Development - Learn GitHub, VS Code, and collaboration
  • Web Fundamentals - Understand hosting, domains, and server workflows
  • Community Building - Create connected digital spaces

Students apply coding skills while experiencing the collaborative, indie web spirit that preceded today's algorithmic platforms.
