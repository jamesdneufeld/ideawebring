Introduction to HTML & CSS Foundational Skills

James' Notion library is the core curriculum for designers learning to build for the web. It is a curated guide for graphic designers, illustrators, and UI/UX designers who need foundational web skills—without becoming developers. What Is a Web Ring?

Back in the early internet, people imagined the web as a giant house, where every website was a room. A web ring was the hallway connecting them—a loop of indie sites with something in common.

There were no algorithms or ads—just curiosity and handcrafted links.

Web rings helped fanfic writers, zine-makers, bloggers, and artists find each other. They were a grassroots way to get discovered before search engines ruled everything.

Today, creators in the indie web, Neocities, and creative coding scenes are bringing them back as a form of digital rebellion. It’s slow, intentional internet—and that’s exactly the energy this student ring channels.

Explore the Ring

Students in both the Bachelor of Design and Interaction Design diploma programs participate in the web ring through a Summer 2026 Coding Mentorship.

The IDEA Web Ring began as a voluntary summer project among passionate students and has now grown into a school-wide initiative to build coding culture at IDEA. What started as a connected network of student-built websites has evolved into an ongoing creative coding playground: a space for experimentation, iteration, indie-web style learning-through-making, and playful technical exploration.

Rather than treating websites as “finished” portfolio pieces, students are encouraged to build evolving digital sketchbooks filled with experiments, mini-pages, responsive résumé pages, interactive studies, animations, text effects, creative coding tests, and modular front-end explorations that grow over time.

The web ring is never truly “complete.” It functions more like a studio-based front-end learning environment where students continuously refine, expand, remix, and experiment with their personal corner of the web.

Connected students have published and linked their personal websites within the web ring. Click any connected site to enter the ring and start exploring.

More badges coming soon across technical skills, community engagement, and creative expression. We’re building a gamified system to celebrate your coding journey—from responsive design mastery to creative experiments.

If you’re not sure what to add yet, start by customizing your fonts and colours and adding your social media links (Behance, Instagram, CodePen, GitHub). Follow each other on CodePen to see what your classmates are building and share inspiration!

Every site reflects the student’s personal style—and this ring is part of James’ ongoing mentorship in web design and coding. Want your site added? Just ask!

The Tools & Skills

As part of the IDEA Web Ring, students get hands-on with the tools and workflows used by professional developers:

  • ✱ Code Editor: VS Code
  • ✱ Version Control: GitHub Desktop
  • ✱ Publishing: GitHub Pages (your site goes live when you push to the repository)
  • ✱ Collaboration: Single shared GitHub repository (practicing commit messages, pushing updates, and managing changes with others)
  • ✱ Design Assets: Font Awesome icons

How Web Rings Work

Each site in the ring includes a shared HTML snippet with navigation links:

  • ◀️ Previous site
  • ▶️ Next site
  • 🌀 Central hub (that’s this page)

This creates a loop you can click through endlessly—like browsing a hand-picked playlist of websites.

No feeds. No doomscrolling. Just a creative coding community.
